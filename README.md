# A [Python] Fisher Information Package

Here we implement a computationally efficient and flexible calculation of the Fisher Information and associated Information Flux.
For full details see https://arxiv.org/abs/1704.05458

## Compatibility

Python 2.7 and 3.0

## How to Install

````PYTHON
pip install rockfish
````
or run 

````PYTHON
python setup.py
````
## Functions Available

Function               | Description
-----------------------|------------
`Fish`		       | Base class. Basic definiton of the Fisher Information with systematics
