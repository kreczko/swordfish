# A [Python] Fisher Information Package

Here we implement a computationally efficient and flexible calculation of the Fisher Information and associated Information Flux.
For full details see https://arxiv.org/abs/1704.05458

NOTE: This package is still in alpha phase and under active development.  Use at your own risk.

Citations:  If you happen to use this software in scientific publications, cite the above paper.


## Compatibility

Tested python 2.7


## How to Install

````PYTHON
python setup.py install
````
